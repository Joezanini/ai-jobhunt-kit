# Grammar & Style Guide

Apply to every resume and cover letter before finalizing. See also `.cursor/rules/public-facing-grammar.mdc`.

## Resumes

1. **Complete sentences** in Summary — limit em dashes to one per paragraph.
2. **Tense**: Present for current role; past for prior roles.
3. **Bullets**: Parallel structure — strong verb first. No first-person in body bullets.
4. **Hyphenation**: AI-assisted, hands-on, cross-functional, open-source.
5. **Oxford comma**; no double spaces.

## Cover Letters

1. Professional tone; 3–5 sentences per paragraph.
2. Do not copy resume bullets verbatim.
3. Opening: role + company in sentence one.
4. Sign-off format:

```
Sincerely,

Your Name
```

## Clickable links

- Plain email and phone in markdown source (converter auto-links).
- `[LinkedIn](url)` for labeled links.
- Never `[email](mailto:...)` when autolink runs on PDF export.

See `docs/CURSOR_RULES.md` and `scripts/batch/convert_to_pdf.py`.
