## Role Summary

Hired as a **Developer Advocate** to teach engineers how to build integrations using ExampleCloud APIs and publish on the **ExampleCloud App Marketplace**. Moved to the marketplace review team while retaining evangelism duties.

Today the role combines:

- **Developer relations** — workshops, blogs, conference talks
- **Open-source stewardship** — ExampleCloud Python SDK on GitHub
- **Partner engineering** — ISV launch support and architecture reviews
- **Documentation** — API guides and migration notes for breaking changes
